# DevProjects

This folder is where I keep the projects I build from DevProjects.

The main reason for it is simple: I want one place in my repo for project-based practice. Instead of just reading or following tutorials, this lets me build things, work through problems, and keep a record of what I’ve made over time.

Each project in here will have its own folder. Some will be small and straightforward, while others may be more involved. Not everything will be perfect or fully polished, but the point of this folder is progress, practice, and getting more comfortable building complete projects from start to finish.

I’ll usually use these projects to work on things like structure, clean code, problem-solving, and getting more used to turning an idea or prompt into something functional.

Over time, this folder should also make it easier to look back and see how my skills, habits, and project quality have improved.
